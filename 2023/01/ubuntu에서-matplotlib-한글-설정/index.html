Ubuntu에서 Matplotlib 한글 설정

January 7, 2023 in Python

Linux 환경을 구축하고 데이터 분석을 진행하다보면 한글 설정이 안되어 있어서 그래프에 한글 표시가 깨져서 나올 때가 있다.


한글 설정 방법

  • 모든 나눔 폰트 설치
  • 터미널에 입력
sudo apt-get install fonts-nanum*
  • font 캐시 갱신
sudo fc-cache -fv
  • 만약 위의 명령어가 되지 않는다면
sudo apt install fontconfig
  • 나눔 font 설치 확인
  • 나눔이 들어간 font 가 확인되면 OK
sudo fc-list


  • 한글 사용 설정
  • 안되면 커널 재시작!
import matplotlib.pyplot as plt

plt.rcParams['font.family']='NanumGothic' # 한글 폰트 사용
  • 한글이 잘 나오는 것을 볼 수 있다!
