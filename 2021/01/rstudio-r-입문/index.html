[Rstudio] R 입문

January 9, 2021 in Rstudio, subcategory

데이터 분석을 하는데 있어서 보통 R과 Python 중 하나를 선택해서 학습합니다.

저 마찬가지로 두 언어 중에 어떤것을 사용할지 많은 고민을 많이 했습니다.

지금은 범용성이 높은 Python과 통계학에 강한 R 중에 Python을 선택했습니다.

그러다가 최근 R의 기본을 익힐 수 있는 강의를 듣게 됐고 이참에 R을 서브로 학습 해야겠다는 생각을 했습니다. (PYTHON도 지금 아주 매우 많이 부족한건 함정)

메인은 Python으로 가져갈 것이기 때문에 간단한 사용법이나 Python 과의 차이점과 비슷한 점 느낀 점 위주로 볼 생각입니다.

분명히 R만의 장점도 있기 때문에 그 점을 파악하고 싶은 생각이 가장 큽니다.

(ADP 실기를 준비하고 있는데 R이 더 유리하다는 평이 있어서 더 궁금하네요)

  • Iris 데이터

R에서는 변수에 값을 입력할 때 = 대신 <- 와 같은 기호를 사용합니다. 물론 = 도 사용할 수 있습니다. 하지만 R만의 규칙이라고하니 <-를 사용해 줍니다.

예를들어

x = 1  #(x)
x <- 1 #(o)

이런식으로 사용합니다. 저는 Rstudio를 사용하는데 Alt + - 를 누르면 바로 입력이 됩니다.

자 이제 데이터를 불러옵니다.

데이터는 분류 문제로 아주 유명한 iris 데이터를 사용합니다.

기본적으로 iris 데이터는 내장되어 있어 불러오는 과정이 필요없지만 그래도 써봅니다.

iris <- read.csv('iris.txt', sep = "") #공백을 기준으로 나누기

Python의 pandas를 쓰는거 같아 익숙합니다.

오른쪽 상단에 iris가 만들어 진게 보이고 150개의 행과 5개의 열로 구성된 것을 볼 수 있습니다.

왼쪽 상단에는 바로 dataframe 으로 볼 수 있게 나오네요.

  • 데이터의 구조를 확인하는 함수
str(iris)

Python의 info() 와 비슷한 역할을 하는 것 같습니다.

각 데이터의 type과 값도 보여줍니다.

요약 통계를 확인할 수 있는 명령어

summary(iris)

각 특성별로 통계치를 볼 수 있고 Species 같은 경우에는 3종류의 붓꽃 종류가 50개씩 들어가 있는 것을 확인 할 수 있습니다.

  • 4가지의 특성을 한번에 나타내는 산점도를 그려봅니다.

(x축은 꽃받침의 길이, y축은 꽃받침의 너비, 색은 붓꽃의 종류, 원의 크기는 꽃잎의 너비)

library(ggplot2) #ggplot2 패키지 불러오기

ggplot(data=iris, mapping = aes(x = Sepal.Length,
                                y = Sepal.Width))+
  geom_point(aes(colour = Species, size=Petal.Width), alpha=0.7)+
  theme_bw() #배경색 흰색으로 변경

간단한(?) 코드로 4가지 변수를 산점도 그래프 하나에 표현했습니다.

  • 특이한 점은 그래프의 속성을 + 로 이어주는 것 입니다.
